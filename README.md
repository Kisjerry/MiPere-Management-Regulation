# 分析

对成员招募的分析（**“招募分析”**）：

​	可以认为，按**成员行为**来思考，社团需要有吸引成员的**社团资产**，才能使得成员愿意加入共读社。

​	共读社可以通过**特殊活动**来产生独特的**社团资产**，这里特指通过**社团资产**的聚合来吸引特定的成员加入。

​	社团成员也是**社团资产**，并且会带有自身资产进入共读社，所以招募社团成员可以积累社团资产，这里可以形成*正反馈循环*  ，社团成员本身就可以带来增长的效应。

​	

对资产累积的分析（**“资产分析”**）：

​	不同的**社团资产**在效应上会有不同的体现，而这种效应的积累则会推动相应的事件和资产产生。

​	为了保证这种效应的积累能够为*长期发展目标* 服务，这里提出：

​		共读社需要不断对*长期发展目标*  做统一的理解和发展，力求跟上时代

​		共读社需要不断分析当前效应积累是否符合*长期发展目标*  并以此调整当前*资产组合*

​	

成员活跃问题的分析（**‘’活跃分析“**）

​	成员的活跃基于**成员行为**，常见为成员依据自身想法做事情。

​	将成员的活跃行为单列视为一个具备生命周期的属性，于是可以根据成员活动的不同生命周期的状态来制定相应策略，根据所以成员的当前状态来制定社团的整体策略。

​	这里展开对活跃问题的两方面讨论：

​		**”活跃分析_1“**：活跃和**社团资产**以及**特殊活动**的关系。

​			**特殊活动**需要调用社团资产和成员来实现活动目的，成员出于自身需要，社团义务和社会压力，需要来参加社团活动，不过这个活跃的程度大多时候受活动本身的限制，并且成员活跃水平会随着活动消失而大大降低。活动结束之后，可以单独计量成员活动的精神资产和物质资产，这些资产伴随社团成员的个人发挥相应效应。

​			**社团资产**具备的效应若被社团成员认识到，则可以认为形成了一个精神资产和物质资产的*有效对* 。于是社团成员能够主动识别情况，并在需要的时候来调用这个*有效对* 以满足需要。称这个使用的过程为*成员社团行为*，也可以将其识别为活跃行为。该活跃行为的生命周期范围是他调用社团资产和满足需求的全过程。

​		**”活跃分析_2“**：活跃的认识问题以及频率与优化。

​			如果社团成员实质是活跃的，但因各种原因而无法被其它成员感知到，从而认为社团并不活跃，可以预见这种消极的精神资产会自我增殖，导致社团崩溃。因此需要设计一个具备感知效应的**社团资产**。可以认为成员对社团的感知有两种来源，一种是主动参与，一种是被动接受：对于前者，成员依据某*有效对* 可以主动调用作为社团资产的其它社团成员，得到了积极响应，由此容易认为社团是活跃的；对于后者，社团成员被主动邀请或受其它**成员行为**的效应影响，于是产生了对社团的新认识，由此认为社团是活跃的。**活跃分析_2a**：“可以发现其中的“主动”部分是促进社团活跃和建立社团活跃的集体认识的关键，于是可以通过促进和增加**活跃分析_1**提到的内容来推动成员的主动。”

​			**活跃分析_2b**：频率以及优化，社团行为等于社团成员和**社团资产**的行为，社团活跃行为来源于**特殊活动**和*成员社团行为* ，且两者都有自己的生命周期，按**活跃分析_2a**操作，可以提高社团活跃行为的频率和数量。据**特殊活动**产生**社团资产**的效应和对*成员社团行为* 的*有效对* 的认识，应合理安排**特殊活动**来增进符合成员想法和需要的**社团资产**及*有效对* 的产出，从而增加自发性的*成员社团行为*，并设计*资产组合* 来使产生这一流程的自组织，形成*正反馈循环*。



# 概述

对共读社成员的分类（**“成员类型”**）：

​	**成员类型_1**，共读社成员：具社团义务，社团的决定和其有关，需要参与社团行政和遵守约定

​	**成员类型_3**，共读社外联：不具社团义务，与共读社保持亲密关系，对其应用（1.0版本还不存在的）外交管理，因为联系紧密，所以可以将其看作**社团资产**

​	**成员类型_3**，共读社校友：同“共读社外联”，不过属于前共读社成员或不活跃的共读社成员。



对成员的自身行为概述（**“成员行为”**）：

​	假设成员聪明，能干，能为自己的选择负责

​	依据自身想法做事情，实现个人目标和理想，获取知识和意见

​	维持自身社会生活，赚钱和养家，以及履行自己社会身份的义务

​	积累社会财富和精神财富，获得维持自己已有生活范围外的资料

​	

对社团资产的概述（**“社团资产”**）：

​	可以被识别为“一个/一件”事物

​	稳定性：可以持续地产生某种效应，持续时间较长

​	可积累：产生的效应可以积累（允许已经积累的效应劣化损失）

​	具有正向的社会效应和对于组织成员来说可以利用

​	社团资产是社团及社团成员，社团及社团成员的精神资产和物质资产

​		精神资产包括，*知识*，*理念*，*社会关系*（暂定）

​		物质资产包括，*能稳定产生效益的事物*，*社会共识的指示物*（如金钱）（暂定）



对特殊活动的概述（**“特殊活动”**）：

​	相较于日常活动，在目的和资源上有明显的边界

​	目的是推动社团某类资产的形成或利用
